# Reusable Prompts

Drop-in prompts you can copy-paste into any chat to kick off a common task. Works with any agent (Claude Code CLI, Cursor, etc.) — these are just text files.

Each `.md` file in this folder is one prompt:
- **Filename** describes what the prompt does
- **File body** is the prompt itself — copy the contents into chat

When to add a new prompt: if you find yourself typing the same instruction more than once or twice, save it here.
